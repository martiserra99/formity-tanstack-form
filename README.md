# Formity + TanStack Form

Example repository showing how to integrate [Formity](https://formity.app) with [TanStack Form](https://tanstack.com/form/latest).

## Overview

This repository demonstrates how Formity and TanStack Form work together. Formity handles the multi-step flow logic — branching, looping, jumping — while TanStack Form handles validation and field state within each step.

## Getting started

Clone the repository.

```bash
git clone https://github.com/martiserra99/formity-tanstack-form
```

Install the dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```
